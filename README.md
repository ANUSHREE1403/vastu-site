## Getting Started

1. Clone the repository
2. Install dependencies for both backend and frontend
3. Set up environment variables
4. Run the development servers

## Development

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm start
```
